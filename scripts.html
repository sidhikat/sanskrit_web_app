Scripts

In this section, you will will learn how to recognize, interpret, and produce the visual symbols that represent Sanskrit sounds, also known as script. There are several scripts that users may learn. These can be found in the Seven different scripts section. The most popular of these scripts is the Devanāgarī script. The next part of this section focuses on teaching the user how to learn Devanāgarī, by practicing the individual consonants and vowel. You can get some practice by clicking on the Learning Devanāgarī section.
